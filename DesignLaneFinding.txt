LaneFinding UseCase:
1. Read in the images/videos
	1.1 return an ordered map {name: image}
2. Loop through each image item in the map
3. Apply GaussianBlur & Canny
4. Apply Hough Transform
5. Apply hough detection which return lines
6. visualize lines on the image -> visualizedImage
7. Save visualizedImage in a folder with the corresponding name
